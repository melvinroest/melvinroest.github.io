Architecture - Overview

The architecture of XIMPEL is as follows:

The XIMPEL web application consists of ximpel.js, with the main application code, and it utilizes jquery.js for some of its functions. Optionally, for touch support, the hammer.js library can be included.

The content of a XIMPEL application is specified via two XML files: playlist.xml (specifying the content and interaction) and config.xml (specifying application settings).

In the playlist, it is possible to define overlays (for interaction), scores and quizzes, and to use a variety of built-in media types. It is also possible to create new, custom, media types.
