Syntax - Audio mediatype

The audio media type allows you to play audio within your ximpel presentation.

playlist.xml

						<ximpel>
						<playlist>
					    <subject>
					        <media>
					            <audio>
					            	<source file="myaudio" extensions="mp3,ogg" types="audio/mp3,audio/ogg" />
					            </audio>
					        </media>
					    </subject>
						</playlist>
						</ximpel>

The file attribute specifies the filename of the audio that is to be played without the extension. The extensions attribute specifies the extensions that are available and the types attribute specifies the mime types corresponding to the extensions listed with the extensions attribute. The browser chooses which of the audio formats is best supported and will play that format.

In the example above we indicated that we want to play an audio file which is stored in two formats: myaudio.mp3 and myaudio.ogg. The types attribute specifies that the mp3 file is of type audio/mp3 and the ogg file is of type audio/ogg. The browser will pick the source file that is actually used.

Note: the first type listed in the types attribute corresponds to the first extension listed in the extensions attribute. The second type in the types attribute to the second extension in the extensions attribute, and so on.

Note: The types attribute is optional and is only used by the browser to make a better decision about which of the source files is most likely to be supported. If you only want to specify a type for some formats but not for all then you should first list the format for which you want to specify the type in the extensions attribute. For instance, if you want to specify the type for the .ogg file but not for the .mp3 file then you can do: 

<source file="myaudio" extensions="ogg,mp3" types="audio/ogg" />



Playing only part of the audio file

In order to start playing at a certain point within the audio file, the startTime attribute can be used. This specifies the startTime in seconds from which to start playing. 
				
						<audio startTime="5">
							<source file="myaudio" extensions="mp3" />
						</audio>
To control how much of the audio file should be played, you can use the duration attribute that is available for all media types. This specifies the number of seconds to play. So if the startTime="30" and the duration="10" then the audio will play from the 30 seconds point of the audio to the 40 seconds point of the audio. 
				
						<audio startTime="30" duration="10">
							<source file="myaudio" extensions="mp3" />
						</video>

Leaving out the duration attribute means it will play until the end of the audio file. Leaving out the startTime attribute means the audio will be played from the beginning.
