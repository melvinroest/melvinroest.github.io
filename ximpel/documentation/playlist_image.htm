Syntax - Picture mediatype

The image media type allows you to show images within your ximpel presentation.

playlist.xml

						<ximpel>
							<playlist>
							    <subject>
							        <media>
										<image src="demo.png" />
							        </media>
							    </subject>
							</playlist>
						</ximpel>

The src attribute specifies the filename of the image that is to be played.


Positioning of the image

The x, y, width and height attributes can be used to position the image at a certain spot within the presentation and force it to have certain dimensions. If you don't set any of these attributes then the image expands as large as possible such that it covers up the entire width and/or height while maintaining aspect ratio. 

						<image src="demo.png" x="200px" y="200px" width="400px" height="400px" />

By setting the x and/or y attributes, the image will have an offset from the top-left of the view area. Using the x and y offsets do not change the default width and height used for the image (ie. the width/height is such that it expands to fill the view-area while maintaining the aspect ratio). This may cause the image to fall partly outside the view-area. Usually you want to provide width/height attributes when using x and y offsets. By specifying the width and/or height attributes you can change the width and height that the image will have.

The x, y, width and height of the image are all based on a 1920x1080 resolution. So the visible area of the presentation lies between x=0 to x=1920 and y=0 to y=1080. Read more about the native XIMPEL resolution here.


Playing the image for a limited time

By default, the image is played indefinitely and would require some user interaction (clicking an overlay for example) to jump to a different part of the presentation. However, the duration attribute is available for every media type and can be used to limit the play time of the image. It specifies the number of seconds the image should play.

This image will play for 5 seconds

<image src="demo.png" duration="5" />
