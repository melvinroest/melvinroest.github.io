Syntax - Questions

During the playback of a media item (video/audio/picture/etc), questions can be asked. This can be done like this:

playlist.xml - a playlist containing two subjects

						<subject id="subject1" leadsTo="subject1">
							<media>
								<picture src="picture1.png">
									<question startTime="5" questionTimeLimit="10" answer="false">
										Is a dog a reptile?
									</question>
									<question startTime="10" questionTimeLimit="10" answer="2">
										Which of the following animals can fly?
										<option>Cow</option>
										<option>Eagle</option>
										<option>Deer</option>
									</question>
								</picture>
							</media>
						</subject>
This will show a picture and after five seconds it will show a true or false question for which there is a 5 seconds time limit. Ten seconds into the video the second question will appear unless the previous question wasn't answered yet (in that case the question will appear as soon as the other question is answered or the timelimit was reached). When the user chooses an answer he will be shown if his/her answer was correct and what the correct answer is.

Note: when no options are specified then the question is assumed to be a true/false question. Otherwise it is a multiple choice question with the given options as possible answer.

Lists of questions

Sometimes you just want to ask a series of questions and don't want to specify the start-time and timelimit for each question indivually. In that case you can use the <questions> element:

playlist.xml

						<questions startTime="1" questionTimeLimit="5">
							<question answer="true">
								In holland it never rains
							</question>
							<question answer="false">
								Does a bicycle have three wheels?
							</question>
							<question questionTimeLimit="10" answer="2">
								Which of the following animals can fly?
								<option>Cow</option>
								<option>Eagle</option>
								<option>Deer</option>
							</question>
						</questions>
The startTime attribute of the question list specifies when the list of questions will start. These questions will appear one after the other. As soon as one questions ends, the next question will be shown. A default time limit for all questions in the series can be set using the questionTimeLimit attribute on the <questions> tag. The time limit can be overwritten per question by using the questionTimeLimit attribute on that specific question tag.
