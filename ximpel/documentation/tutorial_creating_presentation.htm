Tutorial - Creating a complex XIMPEL presentation

In this tutorial it is shown how to create a complex XIMPEL presentation. We are going to make an interactive music sampler! Extensive use of the stopAtSubjectId will be needed in order to accomplish this. Since it is a live (unprepared) coding session, a lot of mistakes will be made as well and it is shown how to detect and deal with these mistakes.

There are 3 parts:

  • Part 1 shows how to gather materials and content, almost nothing about it has to do with XIMPEL.
  • Part 2 shows how to program this content, with XIMPEL, into an interactive music sampler.
  • Part 3 shows the finished XIMPEL application!
