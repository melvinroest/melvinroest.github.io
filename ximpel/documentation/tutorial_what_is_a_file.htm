Examples - Simple Presentation

This page analyzes a XIMPEL presentation created to teach computer illiterate but smart phone savvy people what files, folders and applications are. The content is not too important, the techniques used in it, however, are illustrative in understanding in how to create XIMPEL presentations. Unfortunately, since the images are copyrighted, we cannot give the media assets. However, we did create a YouTube video that shows the presentation and also explains the text written below in much more detail.

The config tag

The config tag allows to enable controls (pause, stop and resume) and enables the display method to be set to overlays.

Overlays

A lot of subjects have overlays. The leadsTo attribute always lead to another subject of which the id has the same name as the leadsTo attribute of the overlay.

Score

Some subjects have a score tag associated to them. If the overlay tag encloses the score tag, then the value of the score tag is only applied when that overlay is clicked. When the score tag is enclosed by the subject, then the value of the score tag is applied when the user arrives at the subject which encloses the score tag.

Questions

While it is possible to use the quiz tag to introduce questions, it is not strictly needed. In the subject remember_what_folders_do_question it is shown how to do a question strictly through overlays and extra subjects for providing feedback.
